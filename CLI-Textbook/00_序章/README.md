# 00. 序章 - CLIの世界へようこそ

## この教科書について

### 対象読者
- コマンドラインインターフェース（CLI）を初めて学ぶ方
- GUIでの操作には慣れているが、CLIの経験がない方
- AI時代のエンジニアリングスキルを身につけたい方
- 作業を自動化して効率を上げたい方

### 学習の進め方
1. **順番に学習**: 章は難易度順に構成されています
2. **実践重視**: 実際に手を動かしながら学んでください
3. **反復練習**: 一度にすべてを覚える必要はありません
4. **安全第一**: 危険なコマンドには警告を付けています

### 教科書の構成

```
00. 序章 ........................ CLIとは何か、なぜ学ぶのか
01. 基礎編 ...................... Windows/macOS/Linuxの基本操作
02. ファイル操作編 .............. ナビゲーション、ファイル管理
03. 応用技術編 .................. パイプ、変数、スクリプト基礎
04. シェルスクリプティング編 .... Bash/PowerShellスクリプト
05. ツール&ユーティリティ編 ..... テキスト処理、Git、パッケージ管理
06. セキュリティ編 .............. ベストプラクティス、危険なコマンド
07. AI時代のCLI編 .............. AIエージェント活用、自動化
08. プラットフォーム固有編 ...... Windows/macOS特有のTips
09. リファレンス編 .............. コマンド一覧、用語集
10. 演習編 ...................... 初級〜上級の実践問題
```

### 推奨学習時間
- **初級（00-02章）**: 1-2週間
- **中級（03-05章）**: 2-3週間
- **上級（06-08章）**: 2-4週間
- **演習と復習**: 継続的に
