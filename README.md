このリポジトリは、CLIを初めて学ぶ方から実務で活用したい方まで、体系的に学べる包括的な教材です。Windows PowerShell、macOS/Linux の両方に対応し、AI時代における CLI スキルの重要性と活用方法も解説しています。

### 特徴

- ✅ **初心者向け**: 基礎から丁寧に解説
- ✅ **実践的**: すぐに使える実例が豊富
- ✅ **クロスプラットフォーム**: Windows/macOS/Linux すべてに対応
- ✅ **AI時代対応**: AIツールとの効果的な連携方法を解説
- ✅ **セキュリティ重視**: 危険なコマンドや安全な使い方を明示

## 💡 学習のコツ

1. **実際に手を動かす**: 読むだけでなく、コマンドを実行する
2. **小さく始める**: 簡単なコマンドから徐々にステップアップ
3. **毎日練習**: 短時間でも継続することが大切
4. **自分の作業に応用**: 日常の繰り返し作業を自動化してみる
5. **失敗を恐れない**: エラーから学ぶことが多い
6. **記録を残す**: よく使うコマンドをメモする

---

## ⚠️ 安全に学ぶために

### 重要な注意事項

1. **練習用フォルダを作成**: 大切なファイルがない場所で練習
2. **削除コマンドは慎重に**: `-i`や`-Confirm`オプションを活用
3. **バックアップを取る**: 重要なデータは必ずバックアップ
4. **信頼できないスクリプトを実行しない**: 内容を確認してから実行
5. **権限を理解する**: `sudo`や管理者権限は必要最小限に

### 絶対に実行してはいけないコマンド

```bash
# 危険！実行禁止！
sudo rm -rf /           # Linux/macOS: システム全削除
Remove-Item -Recurse -Force C:\  # Windows: システム全削除
```

詳しくは[06. セキュリティ編](./CLI-Textbook/06_セキュリティ編/README.md)で解説しています。
